Diese Anleitung führt dich durch die lokale Einrichtung von Seemodo und die Generierung deiner ersten Anwendung.

## Voraussetzungen

Bevor du beginnst, stelle sicher dass du hast:

- **Node.js** Version 20 oder höher
- **pnpm** Paketmanager (empfohlen)
- Mindestens einen KI-API-Schlüssel (Gemini empfohlen für den Einstieg)

## Installation

<Steps>
  <Step title="Repository klonen">
    ```bash
    git clone https://github.com/your-org/seemodo-app.git
    cd seemodo-app
    ```
  </Step>

  <Step title="Abhängigkeiten installieren">
    ```bash
    pnpm install
    ```
  </Step>

  <Step title="Umgebungsvariablen konfigurieren">
    Erstelle eine `.env` Datei im Projekt-Root:

    ```env
    # KI-ANBIETER (mindestens einer erforderlich)
    GEMINI_API_KEY=dein_gemini_api_key           # Empfohlener Standard
    ANTHROPIC_API_KEY=dein_anthropic_api_key     # Optional
    OPENAI_API_KEY=dein_openai_api_key           # Optional
    GROQ_API_KEY=dein_groq_api_key               # Optional (für Kimi K2)

    # SANDBOX-ANBIETER
    SANDBOX_PROVIDER=modal                        # modal, e2b, oder vercel
    MODAL_BACKEND_URL=https://dein-modal.modal.run

    # OPTIONALE DIENSTE
    MORPH_API_KEY=dein_morph_api_key             # Schnelle Code-Anwendung
    PEXELS_API_KEY=dein_pexels_api_key           # Stock-Bilder
    FIRECRAWL_API_KEY=dein_firecrawl_api_key     # Website-Klonen
    ```

    <Warning>
      Committe niemals deine `.env` Datei in die Versionskontrolle. Füge sie zu `.gitignore` hinzu.
    </Warning>
  </Step>

  <Step title="Modal Backend deployen">
    Wenn du Modal als Sandbox-Anbieter verwendest (empfohlen):

    ```bash
    cd modal-backend
    pip install modal
    modal setup
    modal deploy main.py
    ```

    Kopiere die deployed URL und füge sie als `MODAL_BACKEND_URL` zu deiner `.env` hinzu.
  </Step>

  <Step title="Entwicklungsserver starten">
    ```bash
    pnpm dev
    ```

    Die App ist unter `http://app.seemodo.ai` verfügbar.
  </Step>
</Steps>

## Deine erste Generierung

### Option 1: Einfacher Prompt

1. Öffne `http://app.seemodo.ai`
2. Gib einen Prompt ein wie:
   ```
   Eine moderne Task-Management-App mit dunklem Theme,
   die eine Liste von Aufgaben mit Checkboxen,
   Fälligkeitsdaten und Prioritäts-Labels zeigt
   ```
3. Drücke Enter oder klicke den Pfeil-Button
4. Beobachte wie deine App in Echtzeit generiert wird

### Option 2: Visueller Canvas

1. Navigiere zu `http://app.seemodo.ai/start`
2. Nutze den **Studio** Tab um zu konfigurieren:
   - **Modus**: Wireframe oder HiFi (volle Farbe)
   - **Bildschirmgröße**: Desktop, Tablet oder Mobil
3. Gib deinen Prompt im Textfeld ein
4. Klicke **Generieren**
5. Dein Design erscheint auf dem tldraw Canvas

### Option 3: Website klonen

1. Öffne `http://app.seemodo.ai`
2. Füge eine URL statt eines Text-Prompts ein:
   ```
   https://stripe.com/payments
   ```
3. Seemodo scrapt das Design und erstellt es als React-Code neu

## Die Oberfläche verstehen

### Startseite (`/`)

Die minimalistische Startseite für schnelle Prompts:

- Gib beliebigen Text-Prompt oder URL ein
- Erkennt automatisch ob Eingabe eine URL zum Klonen ist
- Leitet zu `/generation` für volle IDE-Erfahrung weiter

### Visueller Canvas (`/start`)

Der vollausgestattete visuelle Designer:

| Komponente | Beschreibung |
|------------|--------------|
| **Linke Seitenleiste** | Prompt-Eingabe, Design-Einstellungen, Generierungs-Steuerung |
| **Canvas** | tldraw Canvas für Wireframes und generierte Screens |
| **Sandbox-Panel** | Live-Vorschau, Terminal, KI-Chat |

### Code IDE (`/generation`)

Vollständige Entwicklungsumgebung mit:

- Datei-Explorer
- Code-Editor
- Live-Vorschau
- KI-Chat
- Konsolen-Logs

## Mit der KI arbeiten

### Das OpenCode-Panel

Sobald eine Sandbox aktiv ist, bietet das OpenCode-Panel direkte KI-Interaktion:

<Tabs>
  <Tab title="Agent-Modus">
    Direkte Coding-Unterstützung. Die KI kann:
    - Dateien lesen und schreiben
    - Terminal-Befehle ausführen
    - Pakete installieren
    - Fehler automatisch beheben
    - Änderungen zu Git committen
  </Tab>
  <Tab title="Plan-Modus">
    Interaktives Brainstorming für komplexe Projekte:
    - KI stellt klärende Fragen
    - Erstellt einen strukturierten Plan
    - Führt den Plan Schritt für Schritt aus
  </Tab>
</Tabs>

### Effektive Prompts

<Accordion title="Gute Prompt-Beispiele">
  ```
  Erstelle eine Benutzerprofilseite mit:
  - Avatar-Upload
  - Editierbaren Name- und Email-Feldern
  - Dark-Mode-Schalter
  - Speichern-Button mit Ladezustand
  Nutze ein modernes kartenbasiertes Layout mit Tailwind CSS
  ```

  ```
  Baue ein Kanban-Board mit drei Spalten:
  Zu erledigen, In Bearbeitung, Erledigt.
  Karten sollen zwischen Spalten ziehbar sein.
  Inkludiere ein Formular zum Hinzufügen neuer Aufgaben.
  ```
</Accordion>

<Accordion title="Tipps für bessere Ergebnisse">
  - Sei spezifisch bei Layout und Styling
  - Erwähne bestimmte UI-Patterns (Karten, Modals, Tabs)
  - Inkludiere Farbpräferenzen oder Themes
  - Referenziere Design-Systeme wenn zutreffend
  - Zerlege komplexe Apps in mehrere Prompts
</Accordion>
